# nerd-fonts-setup
Script for setting up Nerd Fonts on your machine.

### Included Nerd Fonts
- Jetbrains Mono

### Quick Start

```bash
curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/linux-terminal-setup/nerd-fonts-setup/main/setup.sh | bash
```
